# APSU Minigame Studio Template
A starter template for students to begin learning about creating behavior packs and resource packs using the Minecraft Script API.

Resources:
JaylyDev Script API Docs (For version 1.21.130) - https://jaylydev.github.io/scriptapi-docs/1.21.130/
Bedrock Wiki Script Tutorials - https://wiki.bedrock.dev/scripting/scripting-intro
Microsoft Official Script API Reference Documentation - https://learn.microsoft.com/en-us/minecraft/creator/scriptapi/?view=minecraft-bedrock-stable
Blockbench - https://www.blockbench.net/
